# Maze_Example

Пример простой игры лабиринт.  
Управляя игроком нужно дойти до выхода на уровне.  

## Управление

Перемещение:  

 * Вверх - стрелка вверх
 * Вниз - стрелка вниз
 * Влево - стрелка влево
 * Вправо - стрелка вправо
 
Команды:

 * Выход - Q

 * Да - Y
 * Нет - N

Сменить раскладку можно в файле InputConfig.cs

## Карты

Находятся в папке Maps

Условные обозначения

 * Пустая клетка - ' '
 * Стена - '#'
 * Выход(ы) - 'e'
 * Точка старта - 's'

Сменить символы можно в файле MapLegend.cs

Требования к карте - на ней должна быть хотя бы одна точка старта и один выход
